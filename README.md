# Palindrome Checker

A simple palindrome checker built using **HTML, CSS, and JavaScript**.

This app checks whether a word or phrase reads the same forwards and backwards, ignoring spaces, punctuation, and capitalization.

## How to Use

1. Enter text into the input field
2. Click the **Check** button
3. The result will display whether the text is a palindrome

## How It Works

- Removes non-alphanumeric characters
- Converts text to lowercase
- Compares the cleaned string with its reversed version

## Built With

- HTML
- CSS
- JavaScript
